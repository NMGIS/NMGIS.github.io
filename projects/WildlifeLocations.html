Wildlife Locations | ArcGIS Quick Capture


ArcGIS Quick Capture provides a great way to quickly capture data via mobile device. Setup up is quite easy and once it's up-and-running, it operates as smooth as butter. For all my personal projects, I utilize the ArcGIS for Personal Use license. If you have any questions or you would like assistance staging a project like this, please reach out!

Brief Overview of Methods

  1. Create a hosted feature layer, and design the schema
  2. Log into Quick Capture, create and stage a new project | choose "start from existing layers"
  3. Download app on app store, login and select project
  4. Capture data
  5. Create a hosted feature layer view *
  6. Create a web map, add the hosted feature layer
  7. Create a web application which is driven by the above web app

*Note: A hosted feature layer view is used to serve the web application. It is necessary to keep the original hosted feature layer private and un-editable by the public.

Backstory

I hike alot! My dog (Rajah) forces me to get out and hike. In effort to document some of the wildlife, which Rajah doesn't chase away, I decided to stage a Quick Capture project. Quick Capture allows me to take a photo and capture the geolocation of the animal with only a few button clicks. The data is automatically uploaded to my ArcGIS Online account; if I'm offline, it will upload the next time I'm connected to the internet. Ideally, I would like to create a Quick Capture shortcut which captures data with a single button press. I've tried to program this with IOS, but IOS shortcuts feel a little lackluster.
