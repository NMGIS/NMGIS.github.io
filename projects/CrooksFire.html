Crooks Fire dNBR - Google Earth Engine


Using Google Earth Engine, Landsat 9, and NIFC open data, I created an application to show the Normalized Burn Ratio (bNBR) of the Crooks Fire. The Crooks Fire began on 4/18/2022 which was located about 10 miles south of Prescott, Arizona. The fire was 100% contained as of June 24, 2022 which resulted in 9,402 acres of land burnt. The primary fules involved were: Timber(Litter and Understory), Chaparral, and brush. As of 7/2/2022, the cause is still listed as "undetermined".

Methods and Instructions

If you're at all interested in replicating this work, read below. While learning and setting up Google Earth Engine wasn't as easy as I expected; If I wanted to replicate this application for other fires, it would be easy to do so. Why? Google Earth Engine's data catalog is ready and waiting. I've outlined the steps below and have shared my JS code. Note: you can also install GEE api in Python

  1. Create a Google - click "Sign Up" at the top right of page
  2. Navigate to Earth Engine Code Editor
  3. Create a new Cloud Project
  4. Write the Javascript....
    • Some stuff you'll need to change:
    • geojson boundary
    • Landsat images
    • Color scheme - I chose a color scheme which best represented my data
  5. Save and Run the Script
  6. Create the app
